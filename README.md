# Oil Retrieval Robot

## Install

1. Download and install the latest [Arduino IDE](https://www.arduino.cc/en/software).
2. Install the libraries for the sensors using the library manager
   1. VL53L1X by Pololu
   2. SparkFun Qwiic 6DoF - LSM6DS0 by Sparkfun Electronics
